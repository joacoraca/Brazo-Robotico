# Brazo-Robotico
Proyecto de MAPE de la ETRR creado por: Joaquin Racamato, Santiago Besze, Genaro Diaz y Geronimo Singore
Nuestro proyecto en crear un juguete de un brazo robótico a base de fibrofacil con intenciones unicamente de entretenimiento basandonos en el diseño del MeArm.
Dentro de este repositorio van a poder acceder a todo nuestro contenido que vamos haciendo como: la maqueta, su codigo y sus componentes.
